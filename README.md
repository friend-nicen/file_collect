## 上手

### 1.介绍

* 基于uniapp-cli + vue3 + pinia，UI库为uniapp官方组件库
* 当前代码和小程序线上代码同步，效果别无二致

### 2.运行

当前源码模拟了部分接口，部分页面可直接运行看效果

```shell
npm i -D 
npm run dev:mp-weixin
```
